怀念呀，2014/2015我做过的有趣的微信游戏

  • 小玩意儿
2019-01-10

最近无聊整理资料。翻到了很久之前做过的一些微信运营的活动和游戏。

虽然当时比较简单。代码也比较一般。但感觉还是蛮有趣的。

稍微整理了一下。留作纪念吧！最关键的是。这些游戏除了设计。完全是由我个人完成的。包括数据库、接口、前端、微信、发布等等。所有代码相关的。甚至很多游戏人物动作场景都是我找的。额。大多数是从别的游戏里摘取过来的。

小tips：手机观看效果更佳。浏览器模拟也可以。

其实说为谁谁做的为谁谁做的也不尽然。因为大多一个类型好多个都要都用。

宝宝爬 - Baby Go

这个是为郑州万达中心制作的。

这是一个使用手机陀螺仪来左右移动的竖屏游戏。我加上了鼠标按住可以移动的操作。其实这是一个抄袭游戏。当时我把人家的所有图片素材全部抓下来自己实现代码逻辑。嗯。只有图片抄袭而已。。。

在线预览：Demo

模拟微信聊天操作 - Telphone

这个是为雅居乐做的。重阳节的活动。

模拟微信聊天和操作。我在其中实现了更多的细节。细节。。。比如手机信号呀，手机时间呀，手机电量呀，返回对话列表的时候最后对话呀。。。可惜。当时也没人发现。。。

摇苹果 - Shakeapple

这个是为平安做的。

这个不是游戏。只是我感觉这种模式还算好玩。苹果的细节较多。这个其实也没什么好说的。至少当时作为活动用的还是蛮多的。。。

github

转盘抽奖 - Turntable

这个是我自己做的。为之后转盘抽奖做预开发。

但是这不是一般的转盘抽奖。我做的是一个完全随意定制的转盘。可以设置将链接最后的参数参数改为4-10之间的任意数字。多尝试几次试试看。或者。你可以不加参数多刷新几次看看。当然。之后最简单的转盘抽奖用的很多。可是并没有用到这个东西。

https://nostarsnow.github.io/my-game/turntable/?list=7

其他
